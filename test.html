Website logo

 

Use the code below and paste it into your website HTML to get a copy of our logo on your site like this (you can skip steps 1-3 if they're not possible, but you won't get the full effect - you might want to download the image version instead, then):

  1. Download the zip file with the fonts for the logo
  2. Unzip the fonts into a directory on your website
  3. Add the path of the directory into which you've unzipped the fonts into the "Font path" field (e.g. /mycss/myfonts/)
  4. Adjust the size of the logo using the bigger/smaller buttons
  5. Copy the code from the "Logo code" field and add it to your website

    6. To get the full impact and use the proper font, you'll need to download this zip
