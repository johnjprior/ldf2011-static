Leeds PHP: Lithium

21st November 2011, 7:00PM The Wellington

Have you struggled to find a great PHP framework, look no further! Lithium is flexible, powerful, light framework which uses many of PHP 5.3 features to enable you to create a smarter web application.

The evening's presenter is Richard McIntyre, a freelance web developer who has just moved back to the UK from Japan after living there for almost 15 years. He has recently started working with Union of Rad (the guys behind Lithium) to help them create client apps with this fantastic framework.
